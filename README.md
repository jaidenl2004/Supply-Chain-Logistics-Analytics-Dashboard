# Supply-Chain-Logistics-Analytics-Dashboard
Supply Chain Analytics project using Python, SQL, and Power BI. Includes data cleaning, KPIs, carrier performance analysis, cost efficiency metrics, and an interactive logistics dashboard.

This project analyzes 2,000 synthetic shipment records to evaluate carrier efficiency, transit times, cost performance, and delay patterns.

Dashboard KPIs
Total Shipments: 2,000
Average Transit Days: 4.62
Total Shipping Cost: $401.9K
Avg Cost per Mile: 0.18
Delay Rate: 10%

Insights
DHL has the highest delay count (39 shipments)
Amazon Logistics shows longest average transit days
Very Heavy shipments cost most per mile
Cost per mile decreases consistently by weight category
FedEx had the highest total costs

 Tools Used
Python (Pandas, NumPy) for data cleaning
SQLite SQL for aggregations
Power BI for dashboard design
